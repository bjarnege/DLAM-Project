## Dependencies

Run:

```bash
poetry config virtualenvs.in-project true
```

to install virtualenv in the project folder for better ide support

Then install all dependencies:

```bash
poetry install
poetry shell
pip install torch
```

Start a shell session within

```bash
poetry shell
```
